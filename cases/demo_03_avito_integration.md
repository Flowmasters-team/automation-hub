# Демо: Сквозная интеграция Авито → Бот → CRM → Telegram

## Схема потока данных

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────┐     ┌──────────────┐
│   Клиент    │────▶│   AI-бот (FSM)   │────▶│  Битрикс24  │────▶│   Telegram   │
│  на Авито   │     │  TF Neuro Agent  │     │     CRM     │     │  менеджера   │
└─────────────┘     └──────────────────┘     └─────────────┘     └──────────────┘
      │                      │                      │                     │
 Пишет в чат         Квалифицирует          Создаёт сделку      Уведомление с
 объявления          лида по FSM            + контакт            полной сводкой
                     8 состояний            в воронку
```

## Как это работает (пошагово)

### 1. Клиент пишет на Авито
Человек видит объявление, нажимает «Написать продавцу», задаёт вопрос.

### 2. AI-бот мгновенно отвечает (< 3 сек)
- Бот подключён к Авито через TF Neuro (канал `avito`)
- Работает на FSM-промпте с 8 состояниями и 22 сигналами
- Ведёт клиента по воронке: приветствие → квалификация → сбор контакта → запись

### 3. Бот собирает данные
В процессе диалога FSM-автомат заполняет поля:

| Поле | Пример |
|------|--------|
| Услуга | Установка пластиковых окон |
| Город | Сыктывкар |
| Адрес | ул. Ленина, 15 |
| Телефон | +7 912 123-45-67 |
| Имя | Алексей |
| Удобное время | Завтра после 14:00 |
| Комментарий | 3 окна в зал, хотим тёплый профиль |

### 4. Автосоздание сделки в CRM
При завершении диалога tool `dialogueSummary` вызывает:
- **Битрикс24 webhook** → `crm.deal.add` + `crm.contact.add`
- Сделка создаётся в нужной стадии воронки
- Все собранные поля заполняются автоматически
- Источник: «Авито — [название объявления]»

**Пример webhook вызова:**
```json
{
  "action": "http.request",
  "parameters": {
    "url": "https://portal.bitrix24.ru/rest/.../crm.deal.add",
    "method": "POST",
    "body": {
      "fields": {
        "TITLE": "Установка окон — Алексей",
        "STAGE_ID": "NEW",
        "SOURCE_ID": "AVITO",
        "COMMENTS": "$input.summary",
        "CONTACT_ID": "$input.contactId"
      }
    }
  }
}
```

### 5. Уведомление менеджеру в Telegram
Одновременно tool `notify.telegram` отправляет сообщение:

```
🔔 Новая заявка с Авито!

👤 Алексей, +7 912 123-45-67
📍 Сыктывкар, ул. Ленина 15
🪟 Установка окон — 3 шт, тёплый профиль
📅 Хочет завтра после 14:00

💬 Диалог: [ссылка]
📊 CRM: [ссылка на сделку]
```

### 6. Менеджер работает с тёплым лидом
- Вся информация уже собрана
- Сделка в CRM, контакт создан
- Менеджер только звонит и подтверждает замер

## Технический стек

| Компонент | Технология |
|-----------|-----------|
| Мессенджер | Авито (API через TF Neuro) |
| AI-движок | TF Neuro Agent + FSM-промпт |
| CRM | Битрикс24 (REST API webhook) |
| Уведомления | Telegram Bot API |
| Оркестрация | TF Neuro tools (actions) |

## Что даёт клиенту

- **0 минут** на первичную обработку заявки (вместо 15-40 мин)
- **100%** заявок обработаны (включая ночь и выходные)
- **70%+** конверсия из диалога в запись на замер
- **Единая воронка** — все лиды из Авито сразу в CRM
- **Прозрачность** — менеджер видит всё в Telegram мгновенно
