## Источники данных и подключение

| Источник | Метод подключения | Частота |
|----------|------------------|---------|
| Битрикс24 CRM | REST API webhook | Real-time |
| TF Neuro (боты) | API + dashboard | Каждые 5 мин |
| Telegram | Bot API / MCP | Real-time |
| Google Sheets (финансы) | Sheets API | Каждый час |
| Google Calendar | Calendar API | Каждые 15 мин |

## Алерты (автоматические уведомления)

| Триггер | Канал | Кому |
|---------|-------|------|
| Конверсия бота < 50% за день | Telegram | Руководитель |
| Просроченных задач > 5 | Telegram | Менеджер |
| Время ответа > 10 мин | Telegram | Менеджер |
| Выручка за неделю < план на 20% | Telegram | Руководитель |
| Бот не отвечает > 5 мин | Telegram | Техподдержка |

## Стек

- **Grafana** — визуализация и алерты
- **PostgreSQL/ClickHouse** — хранение метрик
- **Python-коннекторы** — сбор данных из CRM, ботов, Sheets
- **Cron** — периодический пул данных
