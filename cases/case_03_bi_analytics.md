# Кейс: BI-аналитика — дашборд с 35 метриками

## Проблема

Компания не видела реальную картину бизнеса в цифрах. Данные были разбросаны: продажи в CRM, заявки в мессенджерах, задачи в трекере, финансы в Excel. Руководитель принимал решения на интуиции, а не на данных. Еженедельные отчёты готовил менеджер вручную — 4 часа каждый понедельник.

## Решение

Развернули Grafana-дашборд, который автоматически собирает данные из всех систем и визуализирует 35 бизнес-метрик в реальном времени.

**Блоки дашборда:**

**Воронка продаж:**
- Входящие заявки по каналам (Авито, Telegram, VK, сайт)
- Конверсия на каждом этапе воронки
- Средний чек и LTV
- Скорость закрытия сделки

**Операционка:**
- Загрузка менеджеров (задачи / завершено / просрочено)
- Время ответа на заявку
- SLA по задачам

**Боты и автоматизации:**
- Количество диалогов бота / день
- Конверсия бота (заявка → запись)
- Процент эскалаций на человека
- Топ причин отказа

**Финансы:**
- Выручка / расходы / маржа
- Unit-экономика по каналам привлечения
- ROI на рекламные каналы

**Источники данных:** Битрикс24 CRM, TF Neuro (боты), Telegram, Google Sheets (финансы).

## Результат

| Метрика | До | После |
|---------|-----|-------|
| Время на отчётность | 4 ч/неделю вручную | 0 (автоматически) |
| Актуальность данных | Раз в неделю | Real-time |
| Видимость проблем | Постфактум (когда уже потеряли) | Мгновенные алерты |
| Принятие решений | На интуиции | На данных |

## Для кого подходит

Компании с оборотом от 1 млн/мес, у которых есть CRM и хотя бы 2-3 канала привлечения. Особенно ценно для компаний с отделом продаж от 3 человек, где важно видеть эффективность каждого менеджера и каждого канала.
