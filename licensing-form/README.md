# Отчёт РАО — автозаполнение (ВГТРК)

GUI-приложение для звукорежиссёров: перетаскиваешь MP3-файлы, вписываешь название передачи — получаешь готовый Excel-отчёт для РАО.

## Быстрый старт (готовый .exe)

1. Скачать `RAO_Report.exe` на рабочий компьютер
2. Запустить
3. Вписать название передачи и дату эфира
4. Перетащить MP3-файлы в окно (или нажать "Добавить файлы")
5. Нажать "Сохранить отчёт (.xlsx)"

Также можно перетащить MP3-файлы прямо на иконку `RAO_Report.exe` — программа откроется с уже загруженными треками.

## Сборка .exe

На любом компьютере с Python 3.10+:

```bash
# Полная сборка (с drag-and-drop)
build.bat

# Упрощённая (если tkinterdnd2 не ставится)
build_simple.bat
```

Результат: `dist/RAO_Report.exe` (~15-20 МБ, один файл, без зависимостей).

## CLI-режим (для продвинутых)

```bash
pip install mutagen openpyxl

# Одна передача
python fill_form.py track1.mp3 track2.mp3 \
    --program "АМВ Вездино Церковь ч1" \
    --date "04.09.25" \
    --month "сентябрь" --year "2025"

# Несколько передач через JSON
python fill_form.py --json report_data.json -o "Отчёт_сентябрь.xlsx"
```

## Колонки отчёта РАО

| № | Колонка | Откуда |
|---|---------|--------|
| 1 | Наименование передачи | вводится вручную |
| 2 | Дата и время выхода в эфир | вводится вручную |
| 3 | Название произведения | из тегов MP3 / имя файла |
| 4 | ФИО композитора | из тегов MP3 (artist) |
| 5 | ФИО автора текста | пусто (инструменталки) |
| 6 | Длительность звучания | из аудиофайла |
| 7 | Количество исполнений | по умолчанию 1 |
| 8 | Общий хронометраж | длительность x кол-во |
| 9 | Жанр произведения | "пьеса" |
| 10 | Исполнитель | "инстр. Анс" |

## Поддерживаемые аудиоформаты

MP3, WAV, FLAC, OGG, AIFF, M4A/AAC, WMA
