# Flowmasters — Automation Hub

Центральный репозиторий команды Flowmasters для проектов по автоматизации.

## Структура

### `/agents` — Мультиагентная система
5 кастомных AI-агентов + оркестратор для параллельной/pipeline разработки:
- **api-designer** — REST API, контракты, Pydantic-схемы
- **backend-dev** — FastAPI, SQLAlchemy, async Python
- **code-reviewer** — безопасность, производительность, архитектура
- **devops** — Docker, CI/CD, деплой
- **tester** — pytest, TDD, контрактные тесты
- `orchestrator.py` — параллельный и pipeline запуск агентов

### `/mcp-bitrix24` — MCP-сервер Битрикс24
Модульный MCP-сервер с **197 инструментами** в 9 доменных модулях:
- CRM (лиды, сделки, контакты, компании, счета, товары, воронки, smart-процессы)
- Задачи и проекты (CRUD, комментарии, чеклисты, время, канбан)
- Диск (хранилища, папки, файлы, версии, права)
- Мессенджер (чаты, сообщения, уведомления, боты)
- Календарь (события, секции, доступность, ресурсы)
- Оргструктура (пользователи, отделы, дерево, присутствие)
- Бизнес-процессы (шаблоны, экземпляры, задания, роботы, триггеры)
- Списки (универсальные списки, поля, элементы)
- Телефония (звонки, записи, статистика, SIP)

### `/prompt-tester` — Платформа тестирования промптов
Веб-приложение для автоматизированного тестирования промптов:
- Загрузка промпта + сценарии (JSON)
- SSE-стриминг прогресса в реальном времени
- PASS/FAIL отчёт по каждому сценарию
- Playground — ручной чат с переключением ролей и пресетами

### `/fsm-pipeline` — FSM-конвейер
Конечные автоматы в промптах — воспроизводимый 5-шаговый процесс для любой ниши:
1. Анализ диалогов → выявление багов и паттернов
2. Проектирование → таблица состояний, сигналы, переходы
3. Калибровка → прогон тестовых сценариев
4. Деплой → через API
5. A/B-анализ → метрики до/после

Примеры: оконная компания, агентство недвижимости, аптечная сеть.

### `/scripts/re-toolkit` — Reverse Engineering Toolkit
Playwright-based инструмент для вскрытия закрытых API:
- Перехват XHR-запросов в SPA
- Автоматическая карта эндпоинтов
- Генерация async Python-клиента из карты API

### `/altschool` — Образовательная платформа
Full-stack приложение с AI-учителями:
- **Backend:** FastAPI, PostgreSQL, JWT, SSE-стриминг
- **4 слоя контекста:** личность → предмет → ученик → сессия
- 3 AI-учителя с уникальными личностями (информатика, математика, физика)
- Docker Compose: app + db + frontend
